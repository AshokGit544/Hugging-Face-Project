I built an enterprise manufacturing document intelligence project in Colab using Hugging Face. I created enterprise-style documents with supplier, plant, equipment, company code, date, and finance context. Then I cleaned the text, converted it into Hugging Face Dataset format, demonstrated tokenization, classification, and token classification, chunked the documents for retrieval, enriched each chunk with business metadata, generated dense embeddings using sentence-transformers, and performed semantic retrieval using cosine similarity. Finally, I summarized the retrieved context using a T5 model. This project reflects how document intelligence, semantic retrieval, and summarization can support enterprise search and knowledge access.
